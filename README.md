# GetHttpDownloadLink

### 功能
- 将QQ旋风，网际快车，迅雷的下载网址转换为Http下载地址

### 结构
- GetHttpDownloadLink  *图形界面*
- DownloadLinkConverter *字符串转换函数类*
- DownloadLinkConverterTest *DownloadLinkTransfer的单元测试*
